# Project 17 – Graph Analytics Platform

Enterprise Graph Analytics Platform using Neo4j Graph Data Science (GDS) for PageRank, Centrality Analysis, Community Detection, Network Intelligence, and Enterprise Analytics.

---

# Overview

This project demonstrates advanced Graph Analytics and Graph Data Science.

The platform provides:

- PageRank Analysis
- Centrality Analysis
- Community Detection
- Network Intelligence
- Graph Performance Analysis
- Relationship Analytics

---

# Business Scenario

Organizations need to analyze:

- Customer Networks
- Social Networks
- Fraud Networks
- Supply Chain Networks
- Knowledge Graphs
- IT Infrastructure Networks

Graph Analytics helps identify influential entities, communities, and hidden relationships.
