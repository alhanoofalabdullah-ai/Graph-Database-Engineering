# Installation Guide

## Navigate to Project

```bash
cd project-17-graph-analytics-platform
```

## Start Neo4j

```bash
docker-compose up -d
```

## Install Dependencies

```bash
pip install neo4j
```

## Run PageRank Analysis

```bash
python python/pagerank_algorithm.py
```

## Open Neo4j Browser

```text
http://localhost:7474
```
