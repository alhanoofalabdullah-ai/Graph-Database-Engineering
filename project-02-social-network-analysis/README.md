# Project 02 – Social Network Analysis

Enterprise Social Network Analysis Platform using Neo4j for relationship mapping, community detection, influence analysis, and network intelligence.

---

# Overview

This project demonstrates graph-based social network analysis.

The platform provides:

- Social Graph Modeling
- Friend Network Analysis
- Community Detection
- Influence Scoring
- Relationship Discovery
- Network Intelligence

---

# Business Scenario

Organizations need to analyze connected communities such as:

- Social Networks
- Professional Networks
- Customer Communities
- Employee Collaboration Networks

Graph databases enable efficient analysis of relationships and influence patterns.
