# Installation Guide

## Navigate to Project

```bash
cd project-10-cybersecurity-attack-graph
```

## Start Neo4j

```bash
docker-compose up -d
```

## Install Dependencies

```bash
pip install neo4j
```

## Run Security Analysis

```bash
python python/attack_path_analyzer.py
```

## Open Neo4j Browser

```text
http://localhost:7474
```
