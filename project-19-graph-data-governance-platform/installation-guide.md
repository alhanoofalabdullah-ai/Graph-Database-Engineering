# Installation Guide

## Navigate to Project

```bash
cd project-19-graph-data-governance-platform
```

## Start Neo4j

```bash
docker-compose up -d
```

## Install Dependencies

```bash
pip install neo4j
```

## Run Governance Dashboard

```bash
python python/governance_scorecard.py
```

## Open Neo4j Browser

```text
http://localhost:7474
```
