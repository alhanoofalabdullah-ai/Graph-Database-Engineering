# Installation Guide

## Navigate to Project

```bash
cd project-09-data-lineage-graph
```

## Start Neo4j

```bash
docker-compose up -d
```

## Install Dependencies

```bash
pip install neo4j
```

## Run Lineage Tracker

```bash
python python/lineage_tracker.py
```

## Open Neo4j Browser

```text
http://localhost:7474
```
