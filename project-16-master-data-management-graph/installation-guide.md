# Installation Guide

## Navigate to Project

```bash
cd project-16-master-data-management-graph
```

## Start Neo4j

```bash
docker-compose up -d
```

## Install Dependencies

```bash
pip install neo4j
```

## Run MDM Platform

```bash
python python/mdm_loader.py
```

## Open Neo4j Browser

```text
http://localhost:7474
```
