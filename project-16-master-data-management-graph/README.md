# Project 16 – Master Data Management Graph

Enterprise Master Data Management (MDM) Graph Platform using Neo4j for Golden Records, Customer 360, Entity Resolution, Data Governance, and Enterprise Data Quality.

---

# Overview

This project demonstrates graph-powered Master Data Management.

The platform provides:

- Customer 360
- Golden Records
- Entity Resolution
- Duplicate Detection
- Data Quality Monitoring
- Master Data Governance

---

# Business Scenario

Organizations store master data across multiple systems:

- ERP
- CRM
- HR Systems
- Procurement Systems
- Financial Systems

Duplicate and inconsistent records create business risks.

Graph databases help unify and govern enterprise master data.
