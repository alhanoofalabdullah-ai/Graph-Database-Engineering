# Installation Guide

## Navigate to Project

```bash
cd project-01-neo4j-fundamentals
```

## Start Neo4j

```bash
docker-compose up -d
```

## Install Dependencies

```bash
pip install neo4j
```

## Run Python Script

```bash
python python/neo4j_connection.py
```

## Open Neo4j Browser

```text
http://localhost:7474
```
