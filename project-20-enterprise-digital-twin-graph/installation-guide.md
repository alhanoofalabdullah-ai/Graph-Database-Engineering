# Installation Guide

## Navigate to Project

```bash
cd project-20-enterprise-digital-twin-graph
```

## Start Neo4j

```bash
docker-compose up -d
```

## Install Dependencies

```bash
pip install neo4j
```

## Run Digital Twin

```bash
python python/digital_twin_loader.py
```

## Open Neo4j Browser

```text
http://localhost:7474
```
