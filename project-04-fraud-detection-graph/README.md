# Project 04 – Fraud Detection Graph

Enterprise Fraud Detection Graph using Neo4j for fraud ring identification, suspicious transaction analysis, anti-money laundering (AML), and financial crime investigations.

---

# Overview

This project demonstrates graph-based fraud detection.

The platform provides:

- Fraud Ring Detection
- Suspicious Transaction Analysis
- Risk Scoring
- AML Analytics
- Investigation Support
- Relationship Intelligence

---

# Business Scenario

Financial institutions process millions of transactions daily.

Fraudsters often operate through:

- Connected Accounts
- Fake Identities
- Transaction Chains
- Money Laundering Networks

Traditional SQL systems struggle to identify hidden relationships.

Graph databases expose fraud networks quickly.
